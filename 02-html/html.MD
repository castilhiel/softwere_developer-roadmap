# HTML home

HTML é a linguagem padrão para páginas WEB.

Com HTML você pode criar seu proprio site.

### Exemplo de da sintaxe do HTML:

```html
    <!DOCTYPE html>
    <html>
        <head>
            <title>Nome do Site<title>
        </head>
        <body>
            <h1>Isso é um cabeçalho</h1>
            <p>Isso é um paragrafo</p>
        </body>
    </html>
```
