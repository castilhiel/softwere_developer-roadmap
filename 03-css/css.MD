# CSS home

CSS é a linguagem que usamos para estilizar um documento HTML.

CSS descreve como os elementos HTML devem ser exibidos.

### Exemplo de da sintaxe do CSS:

```css
body {
  backgrond-color: blue;
}
h1 {
  text-color: green;
}
p {
  text-align: center;
}
```
